# tests/

This directory contains **backend tests only**.

- Repositories (`tests/repositories/`)
- Services (`tests/services/`)
- tRPC routers (`tests/server/routers/`)
- Server auth (`tests/server/auth.test.ts`)
- Middleware (`tests/middleware.test.ts`)
- Lib utilities (`tests/lib/`)
- API routes (`tests/app/api/`)

**Frontend (React component) tests are colocated** — place them next to their source file in `src/`.
