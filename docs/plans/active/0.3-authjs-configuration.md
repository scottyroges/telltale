# Plan 0.3: Auth.js Configuration

**Status:** Not Started
**Depends on:** 0.2 (need Prisma schema with Auth.js models and working database)
**Blocks:** 0.4, 0.5

## Goal

Auth.js v5 fully configured with Prisma adapter, Google OAuth, sign-in/sign-out pages, middleware protecting the `(app)` route group, and session access on both server and client.

## Manual Steps (Before Code)

1. **Create Google OAuth credentials:**
   - Go to https://console.cloud.google.com
   - Create project "Telltale" (or use existing)
   - APIs & Services â†’ Credentials â†’ Create OAuth client ID
   - Type: Web application
   - Authorized JS origins: `http://localhost:3000`
   - Authorized redirect URIs: `http://localhost:3000/api/auth/callback/google`
   - Copy Client ID and Client Secret

2. **Generate AUTH_SECRET:**
   - Run `npx auth secret` or `openssl rand -base64 33`

3. **Update `.env`:**
   - `AUTH_SECRET="<generated>"`
   - `AUTH_GOOGLE_ID="<client-id>"`
   - `AUTH_GOOGLE_SECRET="<client-secret>"`
