# Plan 0.1: Next.js Project Scaffold

**Status:** Not Started
**Depends on:** Nothing — this is the first plan
**Blocks:** 0.2, 0.3, 0.4, 0.5

## Goal

Initialize the project with Next.js 15, TypeScript strict mode, Tailwind CSS, shadcn/ui, and the full directory structure from `docs/architecture/system-overview.md`.

## Manual Steps (Before Code)

None.

## Key Steps

- `npx create-next-app@15 . --typescript --tailwind --eslint --app --src-dir`
- Enable `strict`, `noUncheckedIndexedAccess` in `tsconfig.json`
- `npx shadcn@latest init` + add Button component
- Create directory structure:
  - `src/server/routers/`
  - `src/services/`
  - `src/repositories/`
  - `src/domain/`
  - `src/hooks/`
  - `src/app/(auth)/`
  - `src/app/(app)/dashboard/`
  - `src/app/api/`
- Stub files for `src/server/trpc.ts`, `src/server/auth.ts`, `src/domain/types.ts`
- Update `src/app/layout.tsx` — set title to "Telltale", use Inter font
- Update `src/app/page.tsx` — simple placeholder with Tailwind styles
